# Project Vantage — Inquiry Templates

All inquiries go to **lucaspaganopolisel@gmail.com**.  
Click a link to open a prefilled email draft. Each language section is collapsed by default for readability.

---

## Inquiry Flowchart

```mermaid
flowchart TD
    A[User Needs Help] --> B{What do you need?}

    B -->|Report a bug| BUG[Bug Report Template]
    BUG --> BUG2[Include environment, steps, logs]

    B -->|Request a feature| FEAT[Feature Request Template]
    FEAT --> FEAT2[Describe problem, solution, impact]

    B -->|Security issue| SEC[Security Disclosure Template]
    SEC --> SEC2[Include severity, exploitability, POC]

    B -->|Licensing| LIC[Licensing Inquiry Template]
    LIC --> LIC2[Include intended use, org, scope]

    B -->|Collaboration| COL[Collaboration Proposal Template]
    COL --> COL2[Include project scope, deliverables]

    B -->|Professional contact| PRO[Professional Inquiry Template]
    PRO --> PRO2[Include purpose, context, next steps]
```
